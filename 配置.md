# AI 模型配置

这个文件用于记录发布包里的模型配置方式。你可以随时修改模型名、接入点 ID 或后端地址。

## 当前模型

```ini
ARK_ENDPOINT_ID=ep-20260522175712-qq28w
ARK_MODEL_NAME=deepseek-v3-2-251201
```

## API Key 放置位置

API Key 不应写进 itch.io 静态网页包里。itch.io 上的前端文件会被玩家直接下载和查看，如果把明文 Key 打包进去，任何人都能拿到并消耗额度。

本地开发时，请把 Key 放在项目根目录的 `.env.local`：

```ini
ARK_API_KEY=你的火山方舟 API Key
ARK_ENDPOINT_ID=ep-20260522175712-qq28w
ARK_MODEL_NAME=deepseek-v3-2-251201
PORT=8787
```

`.env.local` 已被 `.gitignore` 排除，不会进入发布包。

## itch.io 发布配置

itch.io 只托管静态网页。如果需要让其他玩家也能实时调用 AI，请把 `server.js` 部署到你自己的后端服务，然后修改 `runtime-config.js`：

```js
window.AI_WEREWOLF_CONFIG = {
  apiBaseUrl: "https://你的后端域名",
  enableRemoteAi: true,
};
```

如果 `apiBaseUrl` 留空，游戏会尝试调用同源 `/api/ai-action`。在 itch.io 静态页面中通常没有这个后端接口，因此需要外部后端代理。

## 修改模型

切换模型时，需要同步修改后端环境变量：

```ini
ARK_ENDPOINT_ID=新的接入点 ID
ARK_MODEL_NAME=新的模型名
```

然后重启后端服务。
